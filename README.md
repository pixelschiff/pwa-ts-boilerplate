# PWA Boilerplate

PWA Boilerplate with webpack, browsersync, sass and bootstrap

- Browsersync with livereload
- Webpack
- Bootstrap 4
- Font Awesome
- ES6
- SASS
- PWA manifests
- Yarn

## Getting Started

Clone it

```sh
git clone https://github.com/pixelschiff/pwa-ts-boilerplate.git
```

Install dependencies
```sh
yarn install / npm install
```

Start server
```sh
npm start
```

Deploy
```sh
npm run build
```
